A Scale Slice is the runtime realization of one [[Scale]] coordinate in an active [[USF Instance Graph]].
Scale slices are runtime integration units that may be simulated in parallel and composed through capability outputs.
They are semantic-runtime slices, not a statement about one mandatory spatial chunking strategy.
Each active slice carries one effective [[Scale Realizer]] binding.
Slice-level invariants include [[Scale Realizer Cardinality]].
