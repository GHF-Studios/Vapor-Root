An Engine Mod is a mod whose parent attachment target is an [[Engine]] fixture or an Engine-facing extension point.

Current framing:

- Engine Mods attach under the selected [[Enginepack]].
- Engine Mods may contribute to the effective Engine identity/fingerprint.
- Engine Mods are part of the Phase 3 proof surface because Phase 3 must prove real Engine-side extension, not only
  first-party stack launch.

See also:

- [[Game Mod]]
- [[Extension Mod]]
- [[Modpack]]
