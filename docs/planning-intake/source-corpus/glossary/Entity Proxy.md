An Entity Proxy is a coupled proxy representation of another entity, used to project behavior, visibility, or both.
Proxy modes include backend-proxy, frontend-proxy, or dual-plane proxy depending on the required behavior surface.
Entity proxies compose with [[Entity Plane Split]], [[Observer-Relative Simulation]], and
[[Portal Traversal Semantics]].
