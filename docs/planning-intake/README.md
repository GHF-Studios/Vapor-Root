# Planning Intake

This stage retains original planning sources exactly as admitted while they
await complete atomization. Their presence here grants no truth, authority,
freshness, or consistency.

- [Canonical planning protocol](../planning/README.md)
- [Corpus register](corpus-register.md)
- [Source corpus](source-corpus/)
- [Atomized candidates](../planning-atomized/README.md)
- [Completed source artifacts](../planning-evidence/source-artifacts/)

Atomization creates independently reviewable candidates in
`planning-atomized/`; it does not rewrite, validate, or reconcile the source.
After complete atomization, the unchanged source moves to
`planning-evidence/source-artifacts/` as permanent provenance.
