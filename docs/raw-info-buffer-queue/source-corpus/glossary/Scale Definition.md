A Scale Definition declares what is semantically meaningful at one [[Scale]] coordinate inside the [[USF Contract]].
This declaration includes the capability types, metric types, and phenomena types recognized at that coordinate, plus
the declared [[Scale Realizer]] type for that coordinate.
Scale-definition declaration and validation rules are governed by the [[Scale Contract]].
