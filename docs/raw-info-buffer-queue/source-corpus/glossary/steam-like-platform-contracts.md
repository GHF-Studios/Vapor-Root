steam-like-platform-contracts is a far-future pressure term for a possible storefront/platform abstraction above or
beside the [[Vapor Ecosystem]].

The idea is to describe the minimum Steam-like identity, ownership, Workshop-style distribution, and publishing
contracts a non-Steam platform would need to implement before Vapor could plausibly target it without becoming a generic
lowest-common-denominator storefront wrapper.

Boundary:
This is not Phase 3 scope.
Current Vapor planning remains [[Steam]]-exclusive and should use Steam directly.
If this layer ever exists, it should not weaken the current Steam-first implementation path.
