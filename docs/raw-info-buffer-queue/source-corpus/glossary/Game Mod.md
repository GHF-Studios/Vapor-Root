A Game Mod is a mod whose parent attachment target is a [[Game]]/`base_mod` fixture or a Game-facing extension point.

Current framing:

- Game Mods attach under the selected [[Gamepack]].
- Game Mods may contribute to the effective Game identity/fingerprint.
- Game Mods are part of the Phase 3 proof surface because Phase 3 must prove real Game-side extension, not only
  first-party stack launch.

- [[Engine Mod]]
- [[Extension Mod]]
- [[Modpack]]
