The Loo Cast Game is the baseline first-party playable/content [[Game]] built on the [[Spacetime Engine]], with the
[[USF]] as the flagship first-party simulation framework.
It is intentionally designed so the player experience can flow toward creator agency through
the [[Player-to-Creator Path]].

Current disambiguation:
`Loo Cast` primarily names the first-party game.
Informally, it may also name the acquired flagship product/project bundle where the game grants access to the
[[Spacetime Engine]] and [[Vapor Ecosystem]] SDK/modding rails, similar to how a first-party Source game can imply
access to Source SDK and Steam ecosystem surfaces.
When precision matters, distinguish the Loo Cast Game from the broader Loo Cast Product/Project bundle.

Current implementation framing:
`base_mod` is the current `Loo Cast Game` [[Reserved Built-In Mod Role]] layer.
It is a required singleton game slot in the first-party stack and is replaceable as a Game-level Vapor concept.
Future `base_mod` extension slots are allowed, but not part of the immediate baseline.
By acquiring Loo Cast, the owner should gain access to the Loo Cast Game, the [[Spacetime Engine]], and the
[[Vapor Ecosystem]]/SDK surface.
