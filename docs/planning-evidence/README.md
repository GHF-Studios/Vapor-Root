# Planning Evidence

This directory contains lifecycle evidence, not planning assertions.

- [`source-artifacts/`](source-artifacts/) retains unchanged sources after
  complete atomization.
- [`extraction-coverage/`](extraction-coverage/) accounts for complete source
  atomization.
- [`assertion-metadata/`](assertion-metadata/) mirrors planning-stage hierarchy
  and stores assertion identity, kind, provenance, temporal scope, coverage,
  and stage.
- [`review-records/`](review-records/) records exhaustive candidate reviews and
  terminal outcomes.

The [canonical planning protocol](../planning/README.md) governs every evidence
type.
